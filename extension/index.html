Spaniel Wallet — browser extension

Same wallet code as spanielwallet.com, packaged as a Chrome / Firefox extension so you can connect to any Solana dApp (MagicEden, Jupiter, Tensor, Drift, pump.fun, etc.) the same way you use Phantom or Solflare. Non-custodial. Your seed phrase never leaves your browser.

Chrome / Edge / Brave / Arc

Manifest V3 · Side-load until Chrome Web Store approval lands.
Download .zip
  1. Unzip the download.
  2. Visit chrome://extensions.
  3. Toggle Developer mode in the top right.
  4. Click Load unpacked, pick the unzipped folder.
  5. Pin the Spaniel Wallet icon (puzzle-piece menu).

Firefox

Manifest V3 · Temporary load until AMO review lands.
Download .zip
  1. Visit about:debugging#/runtime/this-firefox.
  2. Click Load Temporary Add-on…
  3. Pick the downloaded .zip (or any file inside it after unzip).
  4. Temporary add-ons disappear on browser restart. Permanent install lands once AMO approves.
These builds are pre-release. Use a small amount of SOL until they land on the Chrome Web Store + AMO. The same code runs at spanielwallet.com with no install required.
